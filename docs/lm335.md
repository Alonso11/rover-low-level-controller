# Documentación LM335 — Sensor de Temperatura de Precisión

## 1. Descripción General

El LM335 es un sensor de temperatura integrado que funciona como un zener de
2 terminales cuyo voltaje de ruptura es directamente proporcional a la temperatura
absoluta en Kelvin (10 mV/K).

Se usa en el rover para monitorear la temperatura del entorno o del hardware.

## 2. Especificaciones Eléctricas

| Parámetro               | Valor              |
| :---------------------- | :----------------- |
| Tipo de salida          | Analógico (zener)  |
| Sensibilidad            | **10 mV/K**        |
| V_out @ 0 °C (273 K)   | 2731 mV            |
| V_out @ 25 °C (298 K)  | 2981 mV            |
| V_out @ 100 °C (373 K) | 3731 mV            |
| Corriente de operación  | 400 µA – 5 mA      |
| Impedancia dinámica     | < 1 Ω              |
| Precisión (calibrado)   | ±1 °C en 100 °C    |

## 3. Circuito de Polarización

El LM335 requiere una resistencia externa para fijar la corriente de bias.
Con VCC = 5 V, R ≈ 2 kΩ da ~1 mA a 25 °C (salida ≈ 2.98 V), dentro del rango válido.

```
5V ──── R (2 kΩ) ──┬──── A1 (ADC)
                   │
                 LM335 (+)
                 LM335 (−)
                   │
                  GND
```

> La tolerancia de R no es crítica; un valor entre 1 kΩ y 4.7 kΩ funciona correctamente.

## 4. Conexión al Arduino Mega

| LM335    | Pin Arduino Mega | Notas                              |
| :------- | :--------------- | :--------------------------------- |
| (+) OUT  | **A6** (PF6)     | Entrada analógica ADC canal 6      |
| (−) GND  | GND              | Tierra común                       |
| R 2kΩ    | De 5V a A6       | Resistencia de polarización externa|

## 5. Fórmula de Conversión

El ADC del ATmega2560 es de 10 bits con referencia AVCC = 5 V:

```
V_mv      = (adc * 5000) / 1023   // voltaje en mV
T_kelvin  = V_mv / 10             // temperatura en K
T_celsius = T_kelvin − 273        // temperatura en °C
```

Ejemplo a 25 °C:
- ADC ≈ 610 → V = (610 × 5000) / 1023 ≈ 2981 mV
- T_K = 2981 / 10 = 298 K → T_C = 298 − 273 = **25 °C**

## 6. Calibración del Offset

Si la lectura difiere de un termómetro de referencia, ajustar `OFFSET_K`
en `examples/test_lm335_temperature.rs`.

Ejemplo: sensor marca 28 °C, referencia dice 25 °C → `OFFSET_K = -3`.

## 7. Ejemplo de Uso

Ver `examples/test_lm335_temperature.rs`.

Comando de flash:
```bash
RAVEDUDE_PORT=/dev/ttyUSB0 RUSTFLAGS="-C target-cpu=atmega2560" \
  cargo +nightly run --example test_lm335_temperature --release \
  -Zjson-target-spec -Zbuild-std=core
```

Salida esperada a temperatura ambiente:
```
ADC=610 V=2981mV T=298K  T=25.1C
```
